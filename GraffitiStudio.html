Visual Graffiti Prompt Studio Pro

ChatGPT (DALL-E 3) などの画像生成AIで利用できる、高品質なグラフィティ・アートやデザインシート作成用のプロンプトを簡単に生成できるツールです。

⚠️ 注意事項 / 生成のヒント
  • 掲載されているサンプル画像は作例イメージです。画像生成AIの仕様上、プロンプト入力時にまったく同じ画像が再生成されるわけではありません。
  • サンプル画像と近い質感やスタイルの作品を作りたい場合は、生成されたプロンプトテキストと一緒に、本ツールのサンプル画像を参考画像（イメージプロンプト）として生成AI（ChatGPT、Midjourney等）に添付・アップロードして送信することをお勧めします。

1 複数単語一括

複数の英単語を カンマ( , ) で区切って入力します。入力された順に異なるスタイルが自動で割り当てられ、コラージュ風のプロンプトを作成します。

START

2 1単語集中

特定の1単語（自分の名前など）を全50種類以上のスタイルで試したい時に最適です。入力した単語がすべてのスタイルリストに適用されます。

START

3 1形式集中

１つのスタイルを選択し、複数の単語に一括適用します。特定のフォントや色味で統一した単語リストを作りたい場合に便利です。

START

プレビュー画像を確認しながら、特定のスタイルにだけ文字を指定できます。こだわりたい場合に最適です。
