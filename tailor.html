➔ 設定方法・スクリプトの書き方
【設定手順】
1. スプレッドシートを作成し「拡張機能」→「Apps Script」を開く
2. 元のコードを消して以下のコードを貼り付けて保存
function doPost(e) { const sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet(); const data = JSON.parse(e.postData.contents); sheet.appendRow([ new Date(), data.userName, data.shoulder, data.width, data.back, data.total, data.sleeve ]); return ContentService.createTextOutput("OK"); }
3. 右上の「デプロイ」→「新しいデプロイ」を選択
4. 種類を「ウェブアプリ」にし、アクセスできるユーザーを「全員」にしてデプロイ
5. 発行されたURLを上のボックスに貼り付けてください
